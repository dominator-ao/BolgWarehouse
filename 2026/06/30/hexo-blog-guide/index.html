Hexo 博客搭建指南

为什么选择 Hexo

Hexo 是一个快速、简洁且高效的博客框架，基于 Node.js 构建。

优势

  1. 速度快 - 几秒钟内即可生成数百个文件
  2. 支持 Markdown - 写作体验极佳
  3. 一键部署 - 支持 GitHub Pages、Vercel 等
  4. 丰富的主题 - 社区活跃，主题美观
